Guideline
GPS Child Tracker is a real time tracking application that can helps users to get the location of their children.
Allow GPS Child Tracker to send and view SMS messages. This is to ensure that SMS request for location are able to be send and receive by the user
Once the application is ready to use, the user need to register their account.
After that, login page will apper and user need to login.
If the user want to track their children, the user need to go the track page. Click the button provided to request the location of their children.
SMS API will appear, send the text 'on' as provided to the selected numbers.
The user will receive SMS that contain the location of the children. The location provided are in coordinates of latitude and longitude.
The page Map will show the location of the children on the Google Maps view once the request has been made.
